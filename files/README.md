# הכפר היווני בלאצ'י · אתר דילים לאיסתא

אתר דו‑חלקי עם backend אמיתי:

- **`index.html`** — אתר הלקוחות (פרונט): תוכן, תמונות מלון, דילים וטופס השארת פרטים.
- **`admin.html`** — אזור ניהול: דילים, גלריית תמונות, תוכן הדף ופניות.
- **`netlify/functions/api.js`** — ה‑backend. שומר הכל ב‑**Netlify Blobs** (מסד הנתונים של נטליפיי).

הכל משותף בין כל הגולשים — מה שמזינים באדמין מופיע אצל כולם.

## העלאה לנטליפיי — צעד אחר צעד

⚠️ **חשוב:** בגלל שיש כאן Function עם תלות (npm), חובה לפרוס דרך **Git** או **Netlify CLI** — לא דרך גרירת קבצים. גרירה ידנית לא מתקינה את הספריות וה‑backend לא יעבוד.

### דרך א' — GitHub (מומלץ, ללא התקנות במחשב)
1. צרו ריפו חדש ב‑GitHub והעלו אליו את כל הקבצים (אפשר דרך הכפתור "Add file → Upload files" באתר GitHub).
2. היכנסו ל‑[app.netlify.com](https://app.netlify.com) → **Add new site → Import an existing project**.
3. בחרו GitHub ואת הריפו שיצרתם.
4. בהגדרות ה‑build השאירו ריק (אין build command) → **Deploy**.
5. נטליפיי מתקין אוטומטית את `@netlify/blobs`, פורס את ה‑Function ומפעיל את מסד הנתונים. תוך דקה יש כתובת חיה.

### דרך ב' — Netlify CLI (אם נוח לכם בטרמינל)
```bash
npm install -g netlify-cli
cd greek-village
npm install
netlify deploy --prod
```

## אחרי שעלה — מה עושים

1. **כניסה לאדמין:** היכנסו לכתובת `https://<האתר-שלכם>.netlify.app/admin.html`
   הסיסמה: `issta2026` (ראו "אבטחה" למטה כדי לשנות).
2. **גלריית המלון:** לשונית "גלריית המלון" → העלו את תמונות המלון. הן נשמרות במסד הנתונים ומופיעות באתר הלקוחות.
3. **תוכן הדף:** לשונית "תוכן הדף" → ערכו כותרות וטקסטים (יש ברירות מחדל מוכנות).
4. **דילים:** לשונית "דילים" → מלאו פרטים, העלו/בחרו תמונה ולחצו "פרסם דיל".
5. **פניות:** כל מי שמשאיר פרטים בפרונט מופיע בלשונית "פניות", עם כפתור חיוג ו‑WhatsApp.

האתר ללקוחות נמצא בכתובת הראשית `https://<האתר-שלכם>.netlify.app/`.

## תמונות — איך זה עובד
- כל תמונה שמעלים באדמין **מוקטנת אוטומטית בדפדפן** (עד ~1500px) ונשמרת במסד הנתונים. אין שום תלות בשרת חיצוני — התמונות תמיד יופיעו.
- עד שתעלו תמונות משלכם, הפרונט מציג תמונות ברירת מחדל מאתר הכפר היווני, כדי שהאתר לא ייראה ריק.

## אבטחת האדמין
הסיסמה הנוכחית היא הגנה בסיסית בלבד (בצד הלקוח). לשינוי: בקובץ `admin.html`, חפשו את השורה
```js
const ADMIN_PASSWORD = "issta2026";
```
ושנו לסיסמה שלכם.

לאבטחה אמיתית מומלץ אחד מהשניים:
- **Netlify Identity** — מערכת התחברות מובנית של נטליפיי.
- **Password protection** ברמת האתר (Site settings → Access control) — בתוכניות בתשלום.
